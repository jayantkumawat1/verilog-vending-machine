# verilog-vending-machine
A Verilog implementation of a finite state machine (FSM) based vending machine.

# Verilog Vending Machine

This was one of the first projects I built while learning Verilog and finite state machines (FSMs). I wanted to understand how digital logic can be used to model something we see in everyday life, like a vending machine.

The design accepts coins, keeps track of the inserted amount, and dispenses the selected product once the required balance is reached. While building it, I spent a lot of time understanding state transitions, debugging my logic, and learning how sequential circuits behave.

### Features

* FSM-based vending machine
* Coin and balance handling
* Product dispensing logic
* Verilog testbench for simulation

### Files

* `vending_machine.v` – Verilog design
* `vending_machine_tb.v` – Testbench
* `waveform.png` – Simulation waveform

### Tools Used

* Verilog HDL
* Visual Studio Code
* Icarus Verilog
* GTKWave
